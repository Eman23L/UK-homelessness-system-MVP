# Homeless Support Project

A location-based support finder for homelessness services.

## Current MVP

- Scrapes homelessness support data from multiple providers
- Stores service data in a local database
- Geocodes locations
- Provides a Flask API
- Shows nearest support services in a frontend flow:
- category selection
- nearest results
- service details with map

## Tech stack

- Python
- Flask
- SQLite
- JavaScript
- Leaflet
- HTML/CSS

## Project structure

- `parsers/` - scraping logic
- `scripts/` - runners and utilities
- `db/` - database setup and access
- `api/` - Flask API
- `frontend/` - app frontend
- `map/` - older map prototype

## Status

This is currently an MVP under active development.
